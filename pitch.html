The Problem

Silent aspiration kills.
Current tools are episodic.

Dysphagia patients aspirate silently between clinical visits with no detection, no alerts, no protection.

🏥
Current Gold Standard
₹25,000
Cost per Videofluoroscopy (VFSS) session — episodic, radiation-intensive, available only in tertiary hospitals
💀
Mortality Risk

Aspiration pneumonia mortality rate — the direct outcome of undetected silent aspiration events

Zero Protection
24/7
Gap in protection between clinical visits. Aspiration events happen at every meal — 3+ times per day
The Solution

Continuous. Non-invasive.
₹1,000 hardware.

A neck-worn dual-sensor wearable + Android clinical dashboard that never stops monitoring.

🎯
3-Class Real-Time Detection
ESP32 classifies every swallow as SAFE / UNSAFE / COUGH at 10Hz with sub-100ms latency using dual-sensor fusion (IMU + microphone)
📱
Clinical-Grade Android Dashboard
Jetpack Compose + MVVM app with live waveform, session history, PDF export, and caregiver alerts. Works entirely offline — no cloud dependency.
🤖
AI Clinical Assistant
Reads live session data and generates contextual guidance. Issues stop-feeding warnings for non-medical caregivers when risk threshold is exceeded.
🚨
Multi-Layer Alerting
Haptic + visual + SMS fallback after 3 consecutive unsafe events. Bypasses Android background restrictions — alert always reaches the caregiver.
📊
PDF Clinical Reports
Session summaries and trend reports exportable to PDF — designed for Speech-Language Pathologists to review between clinical visits.
